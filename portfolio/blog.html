映像・デザイン・電子工作の制作ログや、考えていること。

ブログの初期設定がまだです

ブログ機能を使うには、Supabase の接続情報を設定してください。手順は portfolio/README.md の「ブログ(Supabase)セットアップ」を参照してください。

  1. Supabase で無料プロジェクトを作成
  2. SQL エディタで posts / news テーブルと RLS を作成(READMEにSQLあり)
  3. Authentication で新規登録を無効化し、自分のユーザーを1つ追加
  4. assets/js/config.js に Project URL と anon key を貼る

※ 設定が終わると、このメッセージは自動的に消えます。
